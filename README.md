# ft_printf

The objective of this project is to recode the printf function. My implementation is 70% of the printf speed

## Mandatory Part

### Converters
* s : ascii strings
* S : unicode strings
* p : pointers
* idD : signed decimal
* oO : octal conversion
* uU : unsigned decimal
* xX : hexadecimal conversion
* c : ascii character
* C : unicode character
### Flags

## Bonuses
* ft_dprintf
* ft_sprintf
* %n : number of characters read by printf
* %f : floats
* %k [+-#] : color option
* %b : binaire
* '*' : wildcards
